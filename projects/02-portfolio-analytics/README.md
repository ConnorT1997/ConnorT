# Portfolio Analytics Toolkit

## Overview
Compute core portfolio statistics and export report-ready tables for investment analysis.

## Goal
This toolkit provides:
- **Risk/Return Metrics**: Calculate portfolio risk and return statistics
- **Drawdown Analysis**: Compute maximum drawdown and drawdown periods
- **Correlation Analysis**: Analyze correlations between assets
- **Tracking Error**: Measure portfolio tracking error against benchmarks
- **Rolling Metrics**: Calculate rolling statistics over time windows
- **Report-Ready Output**: Export tables suitable for reports and presentations

## Tools & Technologies
- **Python**: Core language
- **pandas**: Data manipulation and analysis
- **numpy**: Numerical computations
- **matplotlib**: Visualization
- **Unit-test scaffolding**: Comprehensive testing framework
