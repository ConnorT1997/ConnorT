# Investment News Email Bot

## Overview
Automate a daily investment/economics brief from curated sources with configurable filters.

## Goal
This project aims to create an automated system that:
- Fetches investment and economics news from curated RSS feeds and HTTP sources
- Applies configurable filters to select relevant content
- Delivers a daily brief via email

## Tools & Technologies
- **Python**: Core language for RSS/HTTP fetching and email delivery
- **Environment-based configuration**: Uses `.env` files for sensitive credentials
- **Email delivery**: Automated email sending
